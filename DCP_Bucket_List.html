DCP Bucket List

This is my current project. Originally I was scheduled to be a part of the Disney College Program for Fall of 2020, however due to COVID19 that didn't happen. However, I am still developing this app as a way to track all rides, shows, and restaurants that have been attended in Disney for anyone wishing to accomplish everything in Disney. Currently the only version on Github displays food, however this project is currently in the middle of rewrite to implement more Android features. The image on the right is of the outdated version on Github.

To the left is the current redesign of the project. This uses fragments as the main form of navigation throughout the app for a cleaner experience. Future plans have it to where I will connect to an AWS database, along with increased functionality beyond just food. However, I want an MVP set up before working on new features not seen in the version above. This version is currently not on Git, and I don't think it will be on Git until I get the MVP completed. While this project might look dead on Github, I promise it's not.
