aMAZEing

aMAZEing is my latest semi-complete personal project. It focuses on creating a maze randomly using recursive backtracking. The end goal with this project is to create an AI that can solve the maze. Currently the user is able to set the start and end point of the maze. I plan on possibly coming back and implement heuristic pathfinding.
